## Что делает

Берёт готовый .mp4 файл, извлекает кадры через FFmpeg и строит полный Apple-style сайт где скролл управляет анимацией через `<canvas>`. Эффект как у Apple iPhone teardown — объект разбирается/собирается при прокрутке.

Автоматически включает: animated starscape, annotation cards со snap-stop (скролл замирает на каждой карточке), specs с count-up анимацией, navbar → pill transform, glassmorphism feature cards, loader с progress bar.

**Нужен FFmpeg:** `brew install ffmpeg`

## Установка

```bash
curl -fsSL https://raw.githubusercontent.com/sergeyramas/3d-animation-creator-skill/main/install.sh | bash
```
