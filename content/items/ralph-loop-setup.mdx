## Что делает

Ralph — автономный цикл-агент: ты пишешь ТЗ через `/prd`, режешь его на атомарные задачи через `/ralph`, запускаешь `ralph.sh` — и каждый круг это **новый агент с чистым контекстом**, который берёт одну задачу, пишет код, прогоняет тесты, коммитит и идёт дальше. Цикл сам останавливается, когда невыполненных задач не осталось. Чистый контекст на каждой итерации не даёт модели тупеть к концу большого проекта; память живёт в git-истории, `progress.txt` и `prd.json`.

Сырой Ralph гонит до готового без проверки — это удобно для скаффолда и инфры, но опасно. Этот пакет добавляет **обязательный ревью-гейт**: после прогона ветка не считается готовой, пока её не подписали два слоя — L1 (агентский code-reviewer против ТЗ) и L2 (независимое второе мнение другой моделью, напр. `codex review`). Главная метрика — **цена за принятый результат**: выкидываешь больше половины кругов — луп обходится дороже, чем приносит.

Upstream: [snarktank/ralph](https://github.com/snarktank/ralph) (MIT, паттерн Geoffrey Huntley). Этот пакет добавляет ревью-гейт, русское объяснение и guard «когда НЕ использовать».

## Установка

```bash
curl -fsSL https://raw.githubusercontent.com/sergeyramas/ralph-loop-setup/main/install.sh | bash
```

Или отдай файл `AGENT-INSTALL.md` любому агенту (Claude Code или Codex) — он поставит всё сам.
