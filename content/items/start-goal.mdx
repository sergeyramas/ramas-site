## Проблема

Долгие агентские задачи нельзя стартовать из мутной фразы. Если написать «сделай сайт» или «проведи GAP-анализ», агент может начать выполнять раньше, чем понял рынок, критерии готовности и стоп-условия.

**До:** цель живёт в голове, агент сам додумывает детали, результат трудно проверить.

**После:** одна фраза вроде «стартуем цель» запускает intake: допущение, опрос, brainstorming при необходимости, goal brief и проверяемый блок Done when.

## Что делает

Start Goal превращает сырой запрос в исполняемую цель для Codex Goals. Скилл собирает контекст, ограничения, deliverables, критерии готовности и условия остановки. Для творческих и продуктовых задач он сначала ведёт через brainstorming, а уже потом готовит безопасный запуск `/goal`.

## Установка

```bash
curl -fsSL https://raw.githubusercontent.com/sergeyramas/start-goal-skill/main/install.sh | bash
```
