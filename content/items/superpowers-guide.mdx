## Зачем нужна методология поверх агента

Дефолтный Claude Code на запрос «сделай X» сразу пишет код. Без брейнсторма, без плана, без падающего теста, без верификации. В реальной работе это означает кучу итераций, регрессии и бесконечные правки.

Superpowers закрывает эту дыру 14-ю поведенческими skills. Каждый срабатывает **автоматически** на свою фазу:

* `brainstorming` — перед любой creative work, агент сначала **уточняет**, что ты хочешь
* `writing-plans` — превращает спеку в пошаговый план с TDD
* `subagent-driven-development` — раздаёт задачи параллельным агентам
* `verification-before-completion` — требует evidence перед заявлением «готово»
* `receiving-code-review` — обрабатывает фидбек технически, без performative agreement

И ещё 9 skills на debug, worktrees, ship-and-merge.

## Что в гайде

* Все 14 skills с описанием **на русском**, сгруппированы по фазам (Discovery / Execution / Quality / Ship)
* 4 пути установки: official Anthropic marketplace / superpowers-marketplace / Codex / Gemini CLI
